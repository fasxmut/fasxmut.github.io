utx::npow

utx::npow calculates the result of exp to the power of base, the exp must be an integral type.

utx::npow is an object of class utx::npow_class.

  • It always returns floating vlaue: return_float_type .
  • If the result is overflow of the returned result type, it returns utx::nan_v<return_float_type>.
  • To check if a value is nan, please use utx::is_nan(value) .
  • exp_type to template parameter must be integral type.
  • constexpr support.

Calling Sig 

constexpr auto result = utx::npow<N>(base);	// exp N must be an integral type.

Header 

#include <utxcpp/math.hpp>

Example 

#include <utxcpp/math.hpp>
#include <utxcpp/core.hpp>

int main()
{
	constexpr auto x = utx::npow<utx::u32{3}>(10);	// 1000.0f
	constexpr auto y = utx::npow<3>(10);	// 1000.0f
	utx::print(x, y);
}
