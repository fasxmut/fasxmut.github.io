utfs::perms

utfs::perms class is used to define utfs permission objects in filesystem, doing nothing else. (utfs permission objects)

utfs::perms absolutely supports constexpr.

utfs::perms class is derived from utfs::perm_uct, which is a template-specialized class of utx::class_type. All of utx::class_type member functions are inherited.

Commonly, utx::perm_uct is the same as utx::ux32, so static_assert that utfs::perms is derived from utx::ux32 will be true. 

namespace utfs
{

class perms: public utfs::perm_uct
{
	...
};

}

utfs::perms c++ example 

#include <utxcpp/utfs.hpp>
#include <filesystem>

int main()
{
	constexpr utfs::perms yp = utfs::pmor | utfs::pmow | utfs::pmox;
	static_assert(yp == utfs::pmoall);
	static_assert(yp.std_perms() == std::filesystem::perms::owner_all);
}
