NN Bitwise Shift operators

NN Types Bitwise-Like Left and Right Shift operators: operator==, operator!=

Left Shift: operator<<; Right Shit: operator>>

Utxcpp NN Types Bitwise-Like Left and Right Shift behaves like basic bitwise shift, however, NN Types shift by byte, not by bits. For example, a << 7, shifts left for 7 bytes, not 7 bits. It might should be called bytewise shift, but I still keep the name bitwise.

Shift operators are available for:

  • utx::nn1
  • utx::nn2
  • utx::nn3
  • utx::nn4
  • utx::nn5
  • utx::nn6
  • utx::nn7
  • utx::nn8
  • utx::nn9
  • utx::nn10
  • utx::nn11
  • utx::nn12
  • utx::nn13
  • utx::nn14
  • utx::nn15
  • utx::nn16

c++ example 

#include <utxcpp/core.hpp>
#include <utxcpp/nn.hpp>

int main()
{
	constexpr utx::nn64 a = "030f0";
	static_assert((a<<3) == utx::nn64{"30f0000"});
	static_assert((a>>3) == utx::nn64{"3"});
}
