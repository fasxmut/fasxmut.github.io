utx::thread

utx::thread is used to create a std::thread based thread, which is derived from utx::runnable.

thread.start_thread

thread.start_thread is derived from utx::runnable, please see utx::runnable_base .

thread.start

thread.start is derived from utx::runnable, please see utx::runnable_base .

thread.run_thread

thread.run_thread is derived from utx::runnable, please see utx::runnable_base .

thread.run

thread.run is derived from utx::runnable, please see utx::runnable_base .

thread.join

thread.join is derived from utx::runnable, please see utx::runnable_base .

thread.get_id

thread.get_id is derived from utx::runnable, please see utx::runnable_base .

thread.enable_tcrash

thread.enable_tcrash is derived from utx::runnable, please see utx::runnable_base .
