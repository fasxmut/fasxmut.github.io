cpp 08. auto catch exception

The thread will catch every exception per thread-execution by default, so any exception thrown from user-code method .run() of the task will be caught automatically. 

class some_task: public utx::task
{
public:
	void run() override
	{
		throw std::runtime_error{"test exception"};	// Will be caught by catch by default.
	}
};

method .enable_tcrash(true/false);

However, you can disable the auto exception catch, the method is very simple: call method .enable_tcrash . 

// "tcrash" is disabled, and auto-catch is enabled by default.

thread.enable_tcrash(true);		// Exception will be re-thrown without api catch.
thread.enable_tcrash(false);	// Exception will be not re-thrown without api catch.

Final cpp complete example 

#include <utxcpp/thread.hpp>

class some_task: public utx::task
{
public:
	void run() override
	{
		throw std::runtime_error{"test exception"};	// exception will be auto-proccessed.
	}
};

int main()
{
	utx::jthread thread{some_task{}};
}
