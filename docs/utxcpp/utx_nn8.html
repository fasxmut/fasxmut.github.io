utx::nn8

utx::nn8 - utx::nn_class

utx::nn8 is the eight byte NN type, which is a template class alias. It has 8 characters, can encode 8 byte integral value. The sizeof test result for utx::nn8 is exact 8.

utx::nn8 can make some special features, less chars, more values. It is powerful.
