utx::abstract_string_base .size

.size member function of utx::abstract_string_base

Member function .size() of class utx::abstract_string_base returns the character size in count held by the string. .size is defined by utx::abstract_string_base, and can be used by all its derived classes. including:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string 
virtual size_type size() const;

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str{"Hello c++!!"};
	utx::print(str.size()); // 11
}
