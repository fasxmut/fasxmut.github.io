utx::nn13

utx::nn13 - utx::nn_class

utx::nn13 is the thirteen byte NN type template class alias. It has 13 characters, taken from NN set. It can be used to encode 13 byte integral value. The sizeof test result for utx::nn13 is exact 13. However, because the maxium byte of an integral value in c++ is 8, utx::nn13 can encode 13 byte integral, but can not present it in basic integral. So utx::nn13 uses utx::f32 (floating point) to present its value.

utx::nn13 can make some special encoding features, and more features, that basic integral type can not.
