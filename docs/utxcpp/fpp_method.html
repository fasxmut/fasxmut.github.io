.fpp method

.fpp member function of utx::class_type

utx::class_type::fpp

Method .fpp of utx::class_type is used to get the fixed-point part of a value after floor value.

If a value is negative, its fpp is negative too.

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::fx64 a = 134.221;
	utx::fx64 b = a.fpp();
	utx::print(b);	// 0.221
}
