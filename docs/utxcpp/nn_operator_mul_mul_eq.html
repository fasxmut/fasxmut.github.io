NN operator mul, mul-eq

NN types operator*, operator*=

NN types menber function operator*, operator*= define multiply and multiply-equal operators of NN types. They can be used to multiply one NN object with another NN object, multiply one NN object with a string.

NN types operator*, operator*= are available for all NN types:

  • utx::nn1
  • utx::nn2
  • utx::nn3
  • utx::nn4
  • utx::nn5
  • utx::nn6
  • utx::nn7
  • utx::nn8
  • utx::nn9
  • utx::nn10
  • utx::nn11
  • utx::nn12
  • utx::nn13
  • utx::nn14
  • utx::nn15
  • utx::nn16

c++ example 

#include <utxcpp/core.hpp>

using namespace utx::nn_literals;

int main()
{
	constexpr utx::nn9 a = "Hell";
	constexpr auto b = a * "wo"_nn2 * "noo" * "be";
	utx::nn16 c = "a";
	c *= b;
	utx::print(c, c.value());
}
