.operator<=> and comparison member functions

.operator<=> and comparison member functions of utx::xchar_string

.operator<=> and comparison member functions of class utx::xchar_string are used to compare less, equal or greater result of two string objects.

<=> is the c++ three-way comparison or spaceship operator.

.operator<=> and comparisons are available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1 = "abc";
	utx::string s2 = "abc";
	utx::string s3 = "abd";
	std::cout << (s1==s2) << ' ' << (s1<s3) << ' ' << (s3>s2) << std::endl; //true true true
}
