.operator std_type

.operator std_type member function

.operator std_type member function of utx::xchar_string

.operator std_type is a type operator of c++ class, used to convert utx::string, utx::u8string, or utx::wstring to std::string, std::u8string, or std::wstring etc. Calling .operator std_type is equal to calling .cpp_str, they have the same implementation.

.operator std_type member function is available in:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str = "Hello c++!";
	std::string str2 = str;
	utx::print(str, str2);
}
