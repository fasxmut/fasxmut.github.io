utx::pown

utx::pown calculates the result of exp to the power of base. 

template <utx::real_meric base_type, utx::integral_meric exp_type>
inline constexpr auto pown(const base_type & base_value, const exp_type & exp_value)
{
	using return_float_type = utx::make_float_t<base_type>;
	...
}

utx::pown :

  • It always returns floating vlaue: return_float_type .
  • If the result is overflow of the returned result type, it returns utx::nan_v<return_float_type>.
  • To check if a value is nan, please use utx::is_nan(value) .
  • utx::pown means pow n, exp_type must be std::integral. constexpr support.

utx::pow2, utx::pow3, utx::pow5, utx::pow7

Calculate the result of 2,3,5,7 to the power of base.

The result type is the same as base type.

constexpr support. 

auto result1 = utx::pow2(base_value);

auto result2 = utx::pow3(base_value);

auto result3 = utx::pow5(base_value);

auto result4 = utx::pow7(base_value);

Header 

#include <utxcpp/math.hpp>

Example 

#include <utxcpp/math.hpp>
#include <utxcpp/core.hpp>

int main()
{
	constexpr auto x = utx::pown(10, utx::u32{3}); // 1000
	constexpr auto y = utx::pown(10, 3); // 1000.0f
	utx::print(x, y);
}
