utx::nn16

utx::nn16 - utx::nn_class

utx::nn16 is the sixteen byte template class alias of NN type. utx::nn16 has 16 characters, these characters are taken from NN set. utx::nn16 can encode 16 byte integral value. The sizeof test result for utx::nn16 is exact 16. However, because the maxium byte of an integral value in c++ is 8, utx::nn16 can encode 16 byte integral, but can not present it in basic integral. So utx::nn16 uses utx::f32 (floating point) to present its value.

utx::nn16 can make some special encoding features, and more features, that basic integral type can not.
