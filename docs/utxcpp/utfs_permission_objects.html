utfs permission objects

utfs permission objects define owner/group/others r/w/x access permissions on unix-like systems, some of which might be ignored on windows.

  • utfs::pmnone - no permission
  • utfs::pmor - owner read
  • utfs::pmow - owner write
  • utfs::pmox - owner execute
  • utfs::pmoall - ownerall


  • utfs::pmgr - group read
  • utfs::pmgw - group write
  • utfs::pmgx - group execute
  • utfs::pmgall - group all


  • utfs::pmtr - other read
  • utfs::pmtw - other write
  • utfs::pmtx - other execute
  • utfs::pmtall - other all


  • utfs::pmall - owner, group and other all


  • utfs::pmsu - set-uid
  • utfs::pmsg - set-guid
  • utfs::pmsti - sticky-bit
  • utfs::pmmask - mask - bitwise OR of all above bits.

These permission objects are defined by utfs::perms .

utfs permission objects c++ example 

#include <utxcpp/utfs.hpp>

int main()
{
	static_assert(utfs::pmoall == (utfs::pmor | utfs::pmow | utfs::pmox));
	static_assert(utfs::pmall == (utfs::pmoall | utfs::pmgall | utfs::pmtall));
}
