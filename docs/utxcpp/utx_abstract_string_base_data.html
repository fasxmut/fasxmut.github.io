utx::abstract_string_base .data

.data member function of utx::abstract_string_base

Member function .data of class utx::abstract_string_base returns the underlying raw string data. For utx::string, it returns const char *; for utx::u8string, it returns const char8_t *; for utx::wstring, it returns const wchar_t * . .data is defined by utx::abstract_string_base, and can be used by all its derived classes, including:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string 
virtual const value_type * data() const;

virtual value_type * data();

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str{"Hello c++!!!"};
	utx::print(str, str.data()); // Hello c++!!! Hello c++!!!
}
