utx::task

utx::task is an abstract c++ class, used as the base runnable task for utx::nthread_base, utx::jthread_base etc. utx::task is derived from ux::task_base .

utx::thread is the object-oriented thread design of c++ utxcpp, unlike std::thread. However, utx::thread uses std::thread as its native thread type.

.run()

.run is the only pure virtual method that should be overrided by derived class, .run is used as the real thread callback code.
