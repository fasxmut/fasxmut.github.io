cpp 07. virtual destructor and virtual inheritance

virtual destructors and virtual inheritance, utxcpp c++ thread api

The base classes of all utxcpp thread api are virtual

The next layer classes that are derived from the base classes are virtual inheritance.

So if you want to write task class or thread class to be derived from utxcpp thread api, you do not have to write word virtual any more, they are already virtual.

Final cpp complete example 

#include <utxcpp/thread.hpp>
#include <utxcpp/core.hpp>

class some_task: public utx::task
{
public:
	void run() override
	{
		utx::print("Hello c++!");
	}
};

int main()
{
	some_task task;
	utx::thread thread{task};
}
