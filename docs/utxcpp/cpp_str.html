.cpp_str member function of utx::xchar_string

.cpp_str is a member function of utx::xchar_string, used to get the c++ standard format string. Commonly it gets std::string for utx::string, std::u8string for utx::u8string, and std::wstring for utx::wstring.

.cpp_str member function is available in:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

.cpp_str c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1 = "Hello c++!";
	utx::u8string s2 = u8"Hello c++!";
	utx::wstring s3 = L"Hello c++!";

	std::string str1 = s1.cpp_str();
	std::u8string str2 = s2.cpp_str();
	std::wstring str3 = s3.cpp_str();
}
