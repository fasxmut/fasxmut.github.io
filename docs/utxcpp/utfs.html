Utxcpp UTFS - utx filesystem for std::filesystem.

utfs makes some easy to use utils to std::filesystem. Compared to std::filesystem, the names of utfs are shorter, and the interfaces are more simple. But utfs depends std::filesystem, only c++ standard libraries are used.
