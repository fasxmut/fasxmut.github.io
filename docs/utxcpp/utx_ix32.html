utx::ix32

utx::ix32 - utx::i32

utx::ix32 is the utxcpp integral class type, which is the using typealias of utx::class_type. utx::ix32 is the 32-bit signed integral class type, it has all features of basic 32 bit signed integral, it can be used as int, and has more features.

utx::ix32 - utx::i32, the class type of utx::i32.

utx::ix32 supports all basic operators +, -, *, /, %, &&, || and bitwise operators +, -, *, /, <<, >>, &, |, etc.

Some special member functions for utx::class_type, is available for utx::ix32 too:

operator basic_type () operator()() .value()

Because of the "operator basic_type ()" member function, utx::ix32 can be used to assign to int, then utx::ix32 and int can be doubleaction assigned each other: 

utx::ix32 x = 123;
int y = x;
utx::ix32 z = y;

utx::ix32 also has stream input and output operators: operator<< and operator>>, so std::cout, std::cin, and more other iostreams can be used on utx::ix32.

utx::ix32 c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::ix32 x = 123;
	std::cout << "x=>" << x << std::endl; // x=>123
	utx::print(x, x(), x.value()); // 123 123 123
	utx::print(x << 1); // 246
}
