.empty() utx::xchar_string

.empty() is a member function of utx::xchar_string, used to check if the string is empty.

A string is .empty() only if the size is 0, containing no data.

.empty() member function is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1;
	utx::string s2 = "";
	utx::string s3 = " ";
	utx::string s4 = "c++";
	utx::print(s1.empty(), s2.empty(), s3.empty(), s4.empty()); // true true false false
}
