.push_back member function

.push_back member function of utx::xchar_string

.push_back is a member function of utx::xchar_string, used to push back a value to the end of a string. Because utx::string, utx::u8string and utx::wstring are most commonly implemented string type that are derived from utx::xchar_string, .push_back member function is available for them too.

.push_back member function is available in:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

.push_back c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str{"Hel"};
	str.push_back("l");
	std::cout << str << std::endl; // Hell
}
