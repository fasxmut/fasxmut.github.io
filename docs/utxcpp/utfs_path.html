utfs::path

utfs::path provides a more simple to use interface, compared to std::filesystem::path. utfs::path is implemented with std::filesystem::path. 

namespace utfs
{

class path
{
	...
};

}

utfs::path constructors

There are many constructors of utfs::path, but they are just used to let it accept many different string types, and let the syntax checking pass, all of which share the same interface when it just has one parameter: 

utfs::path{path-to-be-sourced};

utfs::path c++ example 

#include <utxcpp/utfs.hpp>

int main()
{
	utfs::path path{__FILE__};
	utx::print(path);
}
