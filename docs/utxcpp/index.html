Utxcpp Project

Utxcpp

Utxcpp is an easy and simple to use c++ library that makes simple things simple. Utxcpp has some classes, templates and functions that can be used in sycl kernel.

Utxcpp is cross-platform, working on all modern c++ compliant (at least c++23) operating systems. The goal of utxcpp is to make frequently used common things be called simply and directly.
