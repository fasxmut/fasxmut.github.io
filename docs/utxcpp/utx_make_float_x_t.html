utx::make_float_x_t

utx::make_float_x_t - utx::make_float_basic_x_t, utx::make_float_uct_x_t

utx::make_float_x_t is used make a floating point class type from an integral value size1. It will make a lowest requirement for the value of size1.

utx::make_float_x_t related - utx::make_float_basic_x_t, utx::make_float_uct_x_t

The bit size computing:

  • (16, 32] - 32 bit
  • (32, 64] - 64 bit
  • (64, 128] - 128 bit

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	static_assert(utx::same_as<utx::make_float_x_t<63>, utx::fx64>);
	static_assert(utx::same_as<utx::make_float_x_t<64>, utx::fx64>);
	static_assert(utx::same_as<utx::make_float_x_t<65>, utx::fx128>);
}
