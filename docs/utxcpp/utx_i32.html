utx::i32

utx::i32 - utxcpp

utx::i32 is the 32 bit signed integral type of utxcpp c++ api. It guarantees 32 bit signed integral. The type is automatically deduced, which is most likely int, but not always.

cpp example

utx::i32 c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::i32 i = 123;
	utx::print("i=>", i);
}
