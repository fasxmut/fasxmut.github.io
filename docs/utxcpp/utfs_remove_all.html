utfs::remove_all

utfs::remove_all is used to remove single, multiple or hierarchical paths.

It returns count value total files removed. 

namespace utfs
{

template <typename ... path_type_list>
utx::uxmax remove_all(const path_type_list & ... path_list)
{
	...
}

}

utfs::remove_call removes paths hierarchically, such as: 

a/b/c

It can remove "a" directly: 

utfs::remove_all("a"); // OK

utfs::remove_all c++ example 

#include <utxcpp/utfs.hpp>
#include <fstream>

int main()
{
	utfs::mkdir("a/b/c", "d/e", "f/g/h", "i/j");
	{std::ofstream{"i/j/out.txt"} << "Hello c++!" << std::endl;}
	utfs::remove_all("a", "d", "f", "i");
}
