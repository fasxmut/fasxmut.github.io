utx::to_string

utx::to_string is a template function that can convert integral, floating point, single character, or other values to utx::string.

utx::to_string is a template function, that can be overloaded to support more and more data types in the future.

Compared with std::to_string, utx::to_string will convert a single character to a related string, not convert its integral value. 

char x = 97;
utx::ix8 y = 97;
utx::ux8 z = 97;
utx::print(
	utx::to_string(x),	// a	---- ---- !!
	utx::to_string(y),	// 97
	utx::to_string(z),	// 97

	std::to_string(x),	// 97	---- ---- !!
	std::to_string(y),	// 97
	std::to_string(z)	// 97
);

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::ix64 a = 188925;
	utx::string str = utx::to_string(a);
	utx::print(a);
}
