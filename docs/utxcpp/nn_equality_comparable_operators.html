NN Equality Comparable operators

NN Types Equality Comparable operators: operator==, operator!=

operator==, operator!=

NN types equality comparable operators are used to compare two NN type objects, which can be the same NN type, or not the same NN types.

Equality comparable operators are available for:

  • utx::nn1
  • utx::nn2
  • utx::nn3
  • utx::nn4
  • utx::nn5
  • utx::nn6
  • utx::nn7
  • utx::nn8
  • utx::nn9
  • utx::nn10
  • utx::nn11
  • utx::nn12
  • utx::nn13
  • utx::nn14
  • utx::nn15
  • utx::nn16

c++ example 

#include <utxcpp/core.hpp>
#include <utxcpp/nn.hpp>

int main()
{
	constexpr utx::nn32 a = "f30";
	constexpr utx::nn32 b = "f30";
	constexpr utx::nn64 c = "f30";
	constexpr utx::nn32 d = "f3";
	static_assert(a == b);
	static_assert(a == c);
	static_assert(a != d);
}
