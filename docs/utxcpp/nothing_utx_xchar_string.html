.nothing() utx::xchar_string

.nothing() utx::xchar_string

.nothing() is a member function of utx::xchar_string, used to check if a string is empty or holds blank characters.

A string is .nothing(), if:

  • the size is zero.
  • the size is not zero, but only holds characters ' ', '\t', '\r', '\n', '\v', '\f' .

(Two backslash are used to escape one backslash here, by boost quickbook, such as two backslash following n in qbk doc, means one backslash following n in html output.) (Boost QuickBook)

.nothing() member function is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1;
	utx::string s2 = " \n\n\t\n\0\n\n";
	utx::string s3 = "c++";
	utx::print(
		s1.nothing(),		// true
		s2.nothing(),		// true
		s2.empty(),		// false
		s2.size(),		// 5
		s3.nothing()		// false
	);
}
