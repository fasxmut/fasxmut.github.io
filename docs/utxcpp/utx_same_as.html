utx::same_as

utx::same_as - Utxcpp Basic Concepts

utx::same_as is a concept to identify if two types are the same at compile-time.

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	static_assert(utx::same_as<utx::ix32, utx::class_type<i32>>);
}
