utfs::path::stem

utfs::path::stem is a member function of class utfs::path, used to get the basename without extension from filename of a file path.

	...

utfs::path::stem c++ example 

#include <utxcpp/utfs.hpp>
#include <utxcpp/core.hpp>

int main()
{
	utfs::path p1{__FILE__};

	utx::print(p1, p1.stem());
}
