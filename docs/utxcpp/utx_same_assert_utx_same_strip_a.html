utx::same_assert, utx::same_strip_assert

utx::same_assert, utx::same_strip_assert

utx::same_assert and utx::same_strip_assert are used to test if two types are the same or not the same. They are alias template, being useful in compile-time.

utx::same_assert 

template <typename type_xt, typename type_yt, bool same>
class same_assert_class;

static_assert is evaluated by constructor:

  • If same is true: 
static_assert(utx::same_as<type_xt, type_yt>);
  • If same is false: 
static_assert(! utx::same_as<type_xt, type_yt>);

The constructor of utx::same_assert_class is consteval .

utx::same_assert is alias template: 

template <typename type_xt, typename type_yt, bool same = true>
using same_assert = utx::same_assert_class<type_xt, type_yt, same>;

utx::same_assert does not strip anything.

utx::same_strip_assert 

template <typename type_xt, typename type_yt, bool same>
class same_strip_assert_class:
	public utx::same_assert_class<utx::remove_cvref_t<type_xt>, utx::remove_cvref_t<type_yt>, same>;

The constructor of same_strip_assert_class is consteval .

std::remove_cvref_t will be applied to type_xt and type_yt of utx::same_strip_assert, so you do not have to strip them any more.

utx::same_strip_assert is alias template: 

template <typename type_xt, typename type_yt, bool same = true>
using same_strip_assert = same_strip_assert_class<type_xt, type_yt, same>;

c++ example 

#include <utxcpp/core.hpp>
#include <iostream>

int main()
{
	const int & x = 123;
	utx::same_assert<decltype(x), int, false>();

	utx::same_assert<utx::i32, utx::i32>();

	utx::same_strip_assert<decltype(x), int, true>();
	utx::same_strip_assert<decltype(x), int>();

	std::cout << "pass\n";
	utx::print("pass");
}
