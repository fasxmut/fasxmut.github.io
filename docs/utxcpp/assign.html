.assign

.assign member function

.assign member function of utx::xchar_string

.assign is a member function of class utx::xchar_string, used to assign string value to an existed utx::xchar_string object.

.assign equals .operator=, and it just does not return this object.

If the paramater passed to .assign is not type utx::xchar_string<__value_type>, the parameter will be propagated to constructors, and call it.

.assign is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1;
	s1.assign("Hello c++!");
	std::cout << s1 << std::endl; // Hello c++!

	s1.assign(std::string{"Hello"});
	std::cout << s1 << std::endl; // Hello

	utx::string s2 = "c++";
	s1.assign(s2);
	std::cout << s1 << std::endl; // c++
}
