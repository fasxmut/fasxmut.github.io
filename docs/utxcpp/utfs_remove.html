utfs::remove

utfs::remove is used to remove one file or batch files. Every parameter passed to it will be treated as the path to be removed.

It returns false if one path does not exist, otherwise returns true. 

namespace utfs
{

template <typename ... path_type_list>
bool remove(const path_type_list & ... path_list)
{
	...
}

}

utfs::remove does not remove paths hierarchically, every item must be one path without sub-paths.

Such as: 

a/b/c

It can remove "c" if "c" has no sub-paths, but it can not remove "a" directly. 

utfs::remove("c"); // OK, if "c" has no sub-paths.

utfs::remove c++ example 

#include <utxcpp/utfs.hpp>

int main()
{
	utfs::remove("dir1");
	utfs::remove("dir2", "dir3/dir4", "dir3", "dir5", "file1", "file2");
}
