utx::round

utx::round: get the round value of utxpp real meric value. The result type is the same as the value type. 

constexpr inline auto round(const type1 & value);

c++ Example 

#include <utxcpp/math.hpp>

int main()
{
	utx::fx32 d = 32.23f;
	d = utx::round(d); // 32.0f
	utx::fx32 e = 32.6f;
	e = utx::round(e); // 33.0f
}
