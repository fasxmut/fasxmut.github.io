utx::runnable

utx::runnable is used to create a std::thread based runnable, which is derived from utx::runnable_base. utx::runnable is still pure virtual class.

.start_thread

.start_thread is derived from utx::runnable_base, please see utx::runnable_base .

.start

.start is derived from utx::runnable_base, please see utx::runnable_base .

.run_thread

.run_thread is derived from utx::runnable_base, please see utx::runnable_base .

.run

.run is derived from utx::runnable_base, please see utx::runnable_base .

.join

.join is derived from utx::runnable_base, please see utx::runnable_base .

.get_id

.get_id is derived from utx::runnable_base, please see utx::runnable_base .

.enable_tcrash

.enable_tcrash is derived from utx::runnable_base, please see utx::runnable_base .
