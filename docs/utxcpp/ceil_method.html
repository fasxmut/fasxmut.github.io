.ceil method

.ceil member function of utx::class_type

utx::class_type::ceil

Method .ceil of utx::class_type can be used to get the ceil value of integral or floating value of utx::class_type.

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	constexpr utx::ix32 a = 254;
	constexpr utx::fx64 b = 542.445;
	constexpr utx::ix32 c = -254;
	constexpr utx::fx64 d = -542.445;
	utx::print(a.ceil(), b.ceil(), c.ceil(), d.ceil());	// 254 543 -254 -543
}
