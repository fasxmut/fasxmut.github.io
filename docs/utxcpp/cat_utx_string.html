.cat - utx::string

.cat is a member function of utx::shell_string, used to concatenate many strings or many different values that are convertible to utxcpp strings together. 

template <typename ... type_list>
utx::shell_string<value_type> cat(const type_list & ... value_list) const;
  • It is a const member function.
  • It returns non-const non-reference object by value.

Unlike .join member function, .cat just concatenate strings or values together without changing existed string contents, then it returns a new connected string object.

.cat member function is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str = "Hello c++!";
	utx::string str2 = str.cat("c++");
	utx::print("str is", str, '\n', "str2 is", str2);
/*
str is Hello c++!
str2 is Hello c++!c++
*/
}
