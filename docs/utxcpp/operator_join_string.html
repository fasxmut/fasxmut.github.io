operator+=, join string

operator+= is a member function of utx::xchar_string, used to join another string to existed string. It is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::xchar_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string s1 = "Hello";
	s1 += "World";
	utx::print(s1);
}
