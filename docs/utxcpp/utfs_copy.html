utfs::copy

utfs::copy() copies paths from "from" to "to".

default copy options

Try to use the best copy options: utfs::cpopt_best by default. 

namespace utfs
{

void copy(const utfs::path & from, const utfs::path & to)
{
	...
}

}

The default copy options value is bitwise-OR of the following values:

  • utfs::cpopt_update_existing
  • utfs::cpopt_recursive
  • utfs::cpopt_copy_symlinks ---------- Copy symlinks as symlinks, not as the files they point to.
  • utfs::cpopt_create_hard_links ------ Instead of creating copies of files, create hardlinks that resolve to the same files as the originals.

set copy options

  • Copy options cpot can be used to set the copy options. 
namespace utfs
{

void copy(const utfs::path & from, const utfs::path & to, const utfs::copy_options & cpopt)
{
	...
}

}

utfs::copy c++ example 

#include <utxcpp/utfs.hpp>
#include <fstream>
#include <utxcpp/core.hpp>

int main()
{
	utfs::path wkdir = utfs::tmpdir();
	utfs::chdir(wkdir);
	utfs::mkdir("a/b/c", "a/c/b", "a/b/c/d");
	{std::ofstream{"a/b/cpp.txt"} << "Hello, c++!" << std::endl;}
	utfs::copy("a", "x");
	utx::print(utfs::exists(wkdir / "x/b/c"), utfs::exists(wkdir / "x/b/cpp.txt")); // true true
}
