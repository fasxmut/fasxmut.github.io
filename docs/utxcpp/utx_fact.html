utx::fact

utx::fact calculates the factorial result of value. 

template <std::unsigned_meric value_type>
constexpr inline value_type fact(const value_type value)
{
	using return_type = utx::remove_cvref_t<value_type>;
	...
}

utx::fact :

  • If result is less than value, it indicates the result is overflow. When that happens, the result is 3 (There is no any unsigned integral factorial result is 3, so utx::fact uses 3 to indicate overflow).
  • The returned type is same as value type, but the value type must satisfy unsigned. That's one of utx::u8, utx::u16, utx::u32, utx::u64, utx::umax, utx::ux8, utx::ux16, utx::ux32, utx::ux64, utx::uxmax.
  • constexpr support.

Header 

#include <utxcpp/math.hpp>

Example 

#include <utxcpp/core.hpp>
#include <utxcpp/math.hpp>

int main()
{
	constexpr auto result1 = utx::fact<utx::u32>(8);
	constexpr auto result2 = utx::fact<utx::ux32>(8);
	utx::print(result);
}
