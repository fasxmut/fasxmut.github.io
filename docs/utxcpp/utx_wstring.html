utx::wstring

utx::wstring is the string type of wchar_t. utx::wstring is an using alias of template instantiated utx::shell_string, used to create wchar_t strings.

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::wstring str{L"Hello c++!"};
	std::wcout << str << std::endl;
}
