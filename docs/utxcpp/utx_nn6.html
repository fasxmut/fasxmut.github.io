utx::nn6

utx::nn6 - utx::nn_class

utx::nn6 is the six byte template alias of NN class type. It has 6 characters, which are taken from NN set. utx::nn6 can be used to encode a 6 byte integral value. The sizeof test result for utx::nn6 is exact 6.

utx::nn6 can make some special data encoding features and more.
