.join - utx::string

.join is a member function of utx::shell_string, used to join many strings or many different data types that are convertible to utxcpp strings together. 

template <typename ... type_list>
utx::shell_string<char_type> & join(const type_list & ... value_list) & ;	// (1)

template <typename ... type_list>
utx::shell_string<char_type> join(const type_list & ... value_list) && ;	// (2)
  • (1). Calling Lvalue string object join(...) returns a non-const Lvalue reference string object.
  • (2). Calling Rvalue string object join(...) returns a Rvalue (temporary) string object.
  • Because (1) and (2), calling .join(...) of a temporary object will never return a Lvalue reference, then make a non-const Lvalue reference to a temporary object join(...) will report compiling error. 
using namespace utx::string_literals;

utx::string & str1 = "Hello c++!"_us.join(123); // ERROR

utx::string str2 = "Hello c++!"_us.join(123); // OK
utx::print("str2 is", str2); // str2 is Hello c++!123

Unlike .cat member function, .join will append strings or values to the end of the caller string, so the caller string's content will be changed.

.join member function is available for:

  • utx::string
  • utx::u8string
  • utx::u16string
  • utx::u32string
  • utx::wstring
  • utx::shell_string

c++ example 

#include <utxcpp/core.hpp>

int main()
{
	utx::string str = "Hello c++!";
	str.join("c++", 123, true, '.');
	utx::print(str); // Hello c++!c++123true.
}
