utfs::copy_file

utfs::copy_file

utfs::copy_file() copies a file from "from" to "to".

Return true if copied, otherwise return false.

default copy options

Try to use the best copy options: utfs::cpopt_best by default. 

namespace utfs
{

bool copy_file(const utfs::path & from, const utfs::path & to)
{
	...
}

}

The default copy options value is bitwise-OR of the following values:

  • utfs::cpopt_update_existing
  • utfs::cpopt_recursive -------------- Useful for utfs::copy(??), might be ingored by utfs::copy_file(??).
  • utfs::cpopt_copy_symlinks ---------- Copy symlinks as symlinks, not as the files they point to.
  • utfs::cpopt_create_hard_links ------ Instead of creating copies of files, create hardlinks that resolve to the same files as the originals.

set copy options

  • Copy options cpot can be used to set the copy options. 
namespace utfs
{

bool copy_file(const utfs::path & from, const utfs::path & to, const utfs::copy_options & cpopt)
{
	...
}

}

utfs::copy_file c++ example 

#include <utxcpp/utfs.hpp>
#include <fstream>

int main()
{
	utfs::chdir(utfs::tmpdir());
	{std::ofstream{"cpp.txt"} << "Hello, c++!" << std::endl;}
	utfs::copy_file("cpp.txt", "cpp.log");
}
