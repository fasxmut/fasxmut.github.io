Esvcpp Library

Esvcpp

Esvcpp is an Easy, Simple, and Versatile c++ library that makes simple things simple.

Esvcpp is cross-platform, working on all modern c++ compliant (at least c++23) operating systems. The goal of esvcpp is to make frequently used common things be called simply and directly.

Easy

The api interface of esvcpp is very easy to use for c++ programmers.

Simple

The simple things will be kept simple, esvcpp does not solve complicated problems.

Versatile

Esvcpp is not for special domains, it is generic for all domains. Special domains still need special c++ libraries.
