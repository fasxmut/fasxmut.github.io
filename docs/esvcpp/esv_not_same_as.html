esv::not_same_as - Esvcpp Basic Concepts

esv::not_same_as is a concept to identify if two types are not the same at compile-time.

c++ example 

#include <esvcpp/core.hpp>

int main()
{
	static_assert(esv::not_same_as<esv::ix32, esv::i32>);
}
