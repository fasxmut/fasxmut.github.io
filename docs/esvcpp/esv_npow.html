esv::npow

esv::npow calculates the result of exp to the power of base, the exp must be an integral type.

esv::npow is an object of class esv::npow_class.

  • It always returns floating vlaue: return_float_type .
  • If the result is overflow of the returned result type, it returns esv::nan_v<return_float_type>.
  • To check if a value is nan, please use esv::is_nan(value) .
  • exp_type to template parameter must be integral type.
  • constexpr support.

Calling Sig 

constexpr auto result = esv::npow<N>(base);	// exp N must be an integral type.

Header 

#include <esvcpp/math.hpp>

Example 

#include <esvcpp/math.hpp>
#include <esvcpp/core.hpp>

int main()
{
	constexpr auto x = esv::npow<esv::u32{3}>(10);	// 1000.0f
	constexpr auto y = esv::npow<3>(10);	// 1000.0f
	esv::print(x, y);
}
