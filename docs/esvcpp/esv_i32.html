esv::i32

esv::i32 - esvcpp

esv::i32 is the 32 bit signed integral type of esvcpp c++ api. It guarantees 32 bit signed integral. The type is automatically deduced, which is most likely int, but not always.

cpp example

esv::i32 c++ example 

#include <esvcpp/core.hpp>

int main()
{
	esv::i32 i = 123;
	esv::print("i=>", i);
}
