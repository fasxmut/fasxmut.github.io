esv::make_uct_t

esv::make_uct_t is used to get an already defined esv::class_type from a basic type or uct type.

esv::make_uct_t gets one of the folowing types:

  • esv::ix8
  • esv::ix16
  • esv::ix32
  • esv::ix64
  • esv::ixmax
  • esv::ux8
  • esv::ux16
  • esv::ux32
  • esv::ux64
  • esv::uxmax
  • esv::fx32
  • esv::fx64
  • esv::fx128
  • esv::fxmax

Other definition of esv::class_type can not be got with esv::make_uct_t.

esv::make_uct_t Usage 

using target_type = esv::make_uct_t<source_basic_type>;
using target_type2 = esv::make_uct_t<esv::i32>; // Get esv::ix32
