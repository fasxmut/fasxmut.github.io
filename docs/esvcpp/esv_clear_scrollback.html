esv::clear_scrollback

esv::clear_scrollback is an object to clear the scrollback characters. It does not work on all environments. It works on linux at least.

Calling Sig: 

esv::clear_scrollback();

c++ Example 

#include <esvcpp/core.hpp>

int main()
{
	esv::clear_scrollback();
}
