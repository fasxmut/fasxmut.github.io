Lvalue copy constructor: dcpp::shared_device

dcpp::shared_device Lvalue copy constructor: it makes the new created object bind to the existed device.

constructor-11

Calling Sig: 

shared_device(const self_type & other__);

The Lvalue copy constructor will bind to existed device which is held by other device, all dcpp::shared_device objects of the same shared device will share the same device and same reference counter, it will increase the reference counter of all shared objects by one, including the new created object.
