method: deck.slip(...)

deck.slip(...) is a method of class dcpp::shared_deck.

deck.slip(...) starts a duckcpp rendering loop automatically. It has two overloading.

Calling Sig: (overloading-1) 

deck.slip();

overloading-1 starts a default rendering loop.

Calling Sig: (overloading-2) 

deck.slip([]{});

You can insert some extra code to the rendering loop with overloading-2.

For example: 

deck.slip(
	[&]
	{
		auto now = timer->getTime();
	}
);
