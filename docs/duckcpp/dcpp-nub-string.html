dcpp::nub::string

dcpp::nub::string is the char based string using alias of dcpp::nub::basic_string.

Duckcpp forks irrlicht.
dcpp::nub::string is renamed from irr::dcpp::core::stringc.

Method: .data()

Return the c++ raw string: const char *

Method: .str()

Return the c++ standard string: std::string

Method: .operator string

Return The c++ standard string: std::string

It is marked explicit.
