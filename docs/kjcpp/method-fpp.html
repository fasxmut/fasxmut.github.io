.fpp method

.fpp member function of kjcpp::class_type

kjcpp::class_type::fpp

Method .fpp of kjcpp::class_type is used to get the fixed-point part of a value after floor value.

If a value is negative, its fpp is negative too.

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::fx64 a = 134.221;
	kjcpp::fx64 b = a.fpp();
	kjcpp::print(b);	// 0.221
}
