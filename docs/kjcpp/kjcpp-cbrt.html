kjcpp::cbrt

kjcpp::cbrt

Calculate the cube root of value.

Calling Sig: 

constexpr auto result = kjcpp::cbrt(value);

constexpr auto result = kjcpp::cbrt.fn<return_type>(value);

kjcpp::cbrt(value)

  • It always returns a float value.
  • The returned float type is one of kjcpp::f32, kjcpp::f64, kjcpp::f128, kjcpp::fmax, kjcpp::fx32, kjcpp::fx64, kjcpp::fx128, kjcpp::fxmax.
  • constexpr support.

kjcpp::cbrt.fn<return_type>(value)

  • The result is converted to return_type .

Header 

#include <kjcpp/math.hpp>

Example 

#include <kjcpp/core.hpp>
#include <kjcpp/math.hpp>

int main()
{
	constexpr auto x = kjcpp::cbrt(-8); // return kjcpp::f32
	kjcpp::print(x); // -2.000000

	constexpr auto y = kjcpp::cbrt(8.12f); // return kjcpp::f32
	kjcpp::print(y); // 2.009950

	constexpr auto z = kjcpp::cbrt(-8.12); // return kjcpp::f64
	kjcpp::print(z); // -2.009950

	constexpr auto a = kjcpp::cbrt(kjcpp::ix32{8}); // return kjcpp::fx32
	kjcpp::print(a); // 2.000000
}
