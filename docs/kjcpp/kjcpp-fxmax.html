kjcpp::fxmax - kjcpp::fmax

kjcpp::fxmax is the maxium floating point class type. As kjcpp::fxmax is equivalent to kjcpp::fx128 in most cases, they have the same bit size, and both of them are floating point class type. kjcpp::fxmax supports all basic floating point operators, and object-oriented class features.

kjcpp::fxmax - kjcpp::fmax, the class type of kjcpp::fmax.

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::fxmax x = 99.971;
	kjcpp::print(x);
}
