kjcpp::real_meric - Kjcpp Basic Concepts

The concept kjcpp::real_meric<type1> is satisfied if type1 is a real number or type1 is a real kjcpp::class_type.

meric means numeric in kjcpp::real_meric.

"real" means the real number in math.

A real number is a disjunction of unsigned integral, signed integral and floating point number in kjcpp.

Accordingly, a real kjcpp::class_type is a disjunction of unsigned kjcpp::class_type, signed kjcpp::class_type, and floating kjcpp::class_type.

However, kjcpp::class_type is always real.

kjcpp::real_meric is basic and used by kjcpp/types.hpp. To use it, just include: 

#include <kjcpp/types.hpp>

However kjcpp/types.hpp is used by the whole project, so, to use it, including kjcpp/core.hpp works too: 

#include <kjcpp/core.hpp>

kjcpp::unsigned_meric

kjcpp::unsigned_meric<type1> is satisfied if type1 is unsigned integral, or type1 is unsigned kjcpp::class_type.

kjcpp::signed_meric

kjcpp::signed_meric<type1> is satisfied if type1 is signed integral, or type1 is signed kjcpp::class_type.

kjcpp::integral_meric

kjcpp::integral_meric<type1> is satisfied if type1 is integral, or type1 is integral kjcpp::class_type.

kjcpp::floating_meric

kjcpp::floating_meric<type1> is satisfied if type1 is floating integral, or type1 is floating kjcpp::class_type.

Examples

Example 

#include <kjcpp/types.hpp>

template <kjcpp::real_meric type1>
class sun
{
public:
	void operator()() const {}
};

int main()
{
	sun<int>{}();
	sun<kjcpp::u32>{}();
	sun<kjcpp::i32>{}();
	sun<kjcpp::f32>{}();

	sun<kjcpp::ux32>{}();
	sun<kjcpp::ix32>{}();
	sun<kjcpp::fx32>{}();

	class window
	{
	};

	//sun<window>{}(); //error: 'window' does not satisfy 'real_meric'
}

Example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::real_meric auto a = 32; // a is deduced to kjcpp::i32 (aka. int).
	kjcpp::real_meric auto b = 3.2; // b is deduced to kjcpp::f64 (aka. double).
	kjcpp::real_meric auto c = kjcpp::ux64{1280}; // c is deduced to kjcpp::ux64 (kjcpp::class_type).
	kjcpp::real_meric auto d = true; // d is deduced to bool.
	kjcpp::real_meric auto e = 'M'; // e is deduced to kjcpp::i8 (aka. char).
	//kjcpp::real_meric auto f = "pear"; // error: const char * does not satisfy kjcpp::real_meric.

	kjcpp::print(a, b, c, d, e);
}

Example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr auto a = kjcpp::real_meric<int>;
	constexpr auto b = kjcpp::real_meric<kjcpp::u8>;
	constexpr auto c = kjcpp::real_meric<kjcpp::fxmax>;

	class sun {};
	constexpr auto d = kjcpp::real_meric<sun>;

	static_assert(a==true && b==true && c==true && d==false);
}

Example 

#include <kjcpp/core.hpp>

int main()
{
	static_assert(kjcpp::unsigned_meric<kjcpp::ux64>);
	static_assert(kjcpp::signed_meric<kjcpp::ix64>);
	static_assert(kjcpp::integral_meric<kjcpp::ux64>);
	static_assert(kjcpp::integral_meric<kjcpp::ix64>);
	static_assert(kjcpp::floating_meric<kjcpp::fx64>);
}
