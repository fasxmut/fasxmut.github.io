kjcpp::mod2n

template <std::integral T, std::unsigned_integral U>
inline constexpr T kjcpp::mod2n(const T & dividend, const U & n);

Calculate dividend % (2^n)

Divisor is (2^n)

The same as kjcpp::mod, but set the divisor to 2^n. Please read the kjcpp::mod doc page.

If n is zero, it gets zero .

Header 

#include <kjcpp/math.hpp>
