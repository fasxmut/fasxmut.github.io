kjcpp::ix32

kjcpp::ix32 - kjcpp::i32

kjcpp::ix32 is the kjcpp integral class type, which is the using typealias of kjcpp::class_type. kjcpp::ix32 is the 32-bit signed integral class type, it has all features of basic 32 bit signed integral, it can be used as int, and has more features.

kjcpp::ix32 - kjcpp::i32, the class type of kjcpp::i32.

kjcpp::ix32 supports all basic operators +, -, *, /, %, &&, || and bitwise operators +, -, *, /, <<, >>, &, |, etc.

Some special member functions for kjcpp::class_type, is available for kjcpp::ix32 too:

operator basic_type () operator()() .value()

Because of the "operator basic_type ()" member function, kjcpp::ix32 can be used to assign to int, then kjcpp::ix32 and int can be doubleaction assigned each other: 

kjcpp::ix32 x = 123;
int y = x;
kjcpp::ix32 z = y;

kjcpp::ix32 also has stream input and output operators: operator<< and operator>>, so std::cout, std::cin, and more other iostreams can be used on kjcpp::ix32.

kjcpp::ix32 c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::ix32 x = 123;
	std::cout << "x=>" << x << std::endl; // x=>123
	kjcpp::print(x, x(), x.value()); // 123 123 123
	kjcpp::print(x << 1); // 246
}
