kjcpp::throw_if

kjcpp::throw_if is a c++ object of the class template. It has two arguments, if the first param is true, it will throw an exception with the information passed from the second parameter.

Calling Sig: 

kjcpp::throw_if(true_or_false, "the message");

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::throw_if(true, "test exception");
}

#include <kjcpp/core.hpp>

int main()
{
	std::error_code ec;
	kjcpp::throw_if(ec, "test exception");
}
