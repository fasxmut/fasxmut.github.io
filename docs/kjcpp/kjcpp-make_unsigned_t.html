kjcpp::make_unsigned_t

kjcpp::make_unsigned_t - kjcpp::make_unsigned_basic_t, kjcpp::make_unsigned_kct_t

kjcpp::make_unsigned_t is used to get an unsigned integral type from a known ingral type.

That is, it gets kjcpp::ux32 from kjcpp::ix32, kjcpp::ux32; gets kjcpp::u32from kjcpp::i32, kjcpp::u32; ..., etc.

kjcpp::make_unsigned_t never gets kjcpp::overflow_type from an integral type, although it has such as check.

However, kjcpp::make_unsigned_t gets kjcpp::overflow_type, if the source type is not integral type.

kjcpp::make_unsigned_t related - kjcpp::make_unsigned_basic_t, kjcpp::make_unsigned_kct_t

kjcpp::make_unsigned_t Usage 

using target_type = kjcpp::make_unsigned_t<source_inegral_type>;
