kjcpp::i32

kjcpp::i32 - kjcpp

kjcpp::i32 is the 32 bit signed integral type of kjcpp c++ api. It guarantees 32 bit signed integral. The type is automatically deduced, which is most likely int, but not always.

cpp example

kjcpp::i32 c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::i32 i = 123;
	kjcpp::print("i=>", i);
}
