kjcpp::raii_lock

kjcpp::raii_lock is a RAII-only lock template class for a mutex. Its usage is similar to std::lock_guard.

kjcpp::raii_lock is designed for kjcpp::print firstly, but it can be used for general purpose.

Calling Sig: 

std::mutex mutex;
{
	kjcpp::raii_lock<std::mutex> lock{mutex};
}	// Auto-unlock when leaving this scope.

c++ example 

#include <kjcpp/core.hpp>
#include <future>

int main()
{
	std::mutex mutex;
	auto f = std::async(
		[&mutex]
		{
			kjcpp::raii_lock<std::mutex> lock{mutex};
			std::cout << "Hello c++!" << std::endl;
		}
	);
	
	kjcpp::print(mutex, "Hello c++ by", "kjcpp::print!");
}
