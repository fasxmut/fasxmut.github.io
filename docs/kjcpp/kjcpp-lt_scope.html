kjcpp::lt_scope

kjcpp::lt_scope

kjcpp::lt_scope is used to check if the first value is less than the scope [a,b] comprised by the second value and the third value.

Their types must satisfy kjcpp::real_meric.

kjcpp::lt_scope Usage 

constexpr bool result = kjcpp::lt_scope(value, a, b);
