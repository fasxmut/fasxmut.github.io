kjcpp::mtx

kjcpp::mtx - kjcpp::mt

kjcpp::mtx is the char type of kjcpp::class_type, it is 'char', neither 'unsigned char' nor 'signed char'. At first, kjcpp has kjcpp::ux8, which is 'unsigned char', and kjcpp::i8, which is 'signed char', but then 'char' is found useful in some case, so kjcpp::mtx is added for 'char'. All arithmetic and bitwise operators are available for kjcpp::mtx, and it has class features and character features.

kjcpp::mtx - kjcpp::mt, the class type of kjcpp::mt.

  • .operator()()
  • .operator basic_type ()
  • .value()
  • .value_ref()

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::mtx x = 'H';
	kjcpp::print(x, x(), x.value()); // H H H
}
