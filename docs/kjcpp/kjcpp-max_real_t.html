kjcpp::max_real_t

kjcpp::max_real_t is used to get the lowest maximum type of two real types.

For example, it gets kjcpp::ix64 from kjcpp::u32 and kjcpp::ix16; gets kjcpp::ix32 from kjcpp::ix32 and kjcpp::u8.

However, kjcpp::max_real_t gets kjcpp::overflow_type, if the converter have a chance to make overflow values. Such as it gets kjcpp::overflow_type from kjcpp::u64 and kjcpp::i32.

kjcpp::max_real_t Usage 

using target_type = kjcpp::max_real_t<real_type1, real_type2>;
