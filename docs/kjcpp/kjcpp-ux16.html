kjcpp::ux16 - kjcpp::u16

kjcpp::ux16 is 16-bit unsigned integral type, just like unsigned short int, but kjcpp::ux16 is a class type, aka kjcpp::class_type, which makes kjcpp::ux16 have advanced object-oriented features, such as member functions, and let it extendible for more features.

kjcpp::ux16 - kjcpp::u16, the class type of kjcpp::u16.

  • .operator()()
  • .operator basic_type ()
  • .value()
  • .value_ref()

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::ux16 x = 65535u;
	kjcpp::print(x, x(), x.value()); // 65535 65535 65535
}
