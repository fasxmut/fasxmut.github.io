kjcpp::make_signed_t

kjcpp::make_signed_t - kjcpp::make_signed_basic_t, kjcpp::make_signed_kkt_t

kjcpp::make_signed_t is used to get a signed integral type from a known source integral type.

That is, it gets kjcpp::ix32 from kjcpp::ux16, kjcpp::ix32; gets kjcpp::i32 from kjcpp::u16, kjcpp::i32; gets kjcpp::i64 from kjcpp::u32, kjcpp::i64; ... etc.

If the source integral type is greater than or equal to kjcpp::u64 or kjcpp::ux64, it gets kjcpp::overflow_type.

If the source type is not kjcpp::integral type, it gets kjcpp::overflow_type too, instead of compile errors.

kjcpp::make_signed_t related - kjcpp::make_signed_basic_t, kjcpp::make_signed_kkt_t

kjcpp::make_signed_t Usage 

using target_type = kjcpp::make_signed_t<source_integral_type>;
static_assert(! kjcpp::is_same_v<target_type, kjcpp::overflow_type>);
