kjcpp::make_kkt_t

kjcpp::make_kkt_t is used to get an already defined kjcpp::class_type from a basic type or kkt type.

kjcpp::make_kkt_t gets one of the folowing types:

  • kjcpp::ix8
  • kjcpp::ix16
  • kjcpp::ix32
  • kjcpp::ix64
  • kjcpp::ixmax
  • kjcpp::ux8
  • kjcpp::ux16
  • kjcpp::ux32
  • kjcpp::ux64
  • kjcpp::uxmax
  • kjcpp::fx32
  • kjcpp::fx64
  • kjcpp::fx128
  • kjcpp::fxmax

Other definition of kjcpp::class_type can not be got with kjcpp::make_kkt_t.

kjcpp::make_kkt_t Usage 

using target_type = kjcpp::make_kkt_t<source_basic_type>;
using target_type2 = kjcpp::make_kkt_t<kjcpp::i32>; // Get kjcpp::ix32
