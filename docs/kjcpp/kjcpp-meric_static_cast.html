kjcpp::meric_static_cast

kjcpp::meric_static_cast is used to convert values between different integral, floating point, or character types. kjcpp::meric_static_cast is the static_cast for kjcpp::real_meric types.

kjcpp::meric_static_cast: 

template <kjcpp::real_meric target_type>
constexpr inline auto meric_static_cast = kjcpp::meric_static_cast_class<target_type>{};

Calling Sig: 

#include <kjcpp/types.hpp>

constexpr auto value2 = kjcpp::meric_static_cast<target_type>(value__);

Requirements

  • Both target_type and the type of value__ must satisfy kjcpp::real_meric, aka. they must be arithmetic number or kjcpp::class_type of arithmetic number.
  • If the target type is non-const Lvalue reference type, the nude type of source value and the nude type of target_type must be the same type, aka. requires: kjcpp::same_as<kjcpp::remove_cvref_t<target_type>, kjcpp::remove_cvref_t<source_type> .

The following types are valid source and target types:

  • kjcpp::real_number types:
    • kjcpp::mb
    • kjcpp::u8
    • kjcpp::u16
    • kjcpp::u32
    • kjcpp::u64
    • kjcpp::umax
    • kjcpp::i8
    • kjcpp::i16
    • kjcpp::i32
    • kjcpp::i64
    • kjcpp::imax
    • kjcpp::f32
    • kjcpp::f64
    • kjcpp::f128
    • kjcpp::fmax
    • char
    • unsigned char
    • char8_t
    • char16_t
    • char32_t
    • wchar_t
  • kjcpp::real_kct types:
    • kjcpp::mbx
    • kjcpp::ux8
    • kjcpp::ux16
    • kjcpp::ux32
    • kjcpp::ux64
    • kjcpp::uxmax
    • kjcpp::ix8
    • kjcpp::ix16
    • kjcpp::ix32
    • kjcpp::ix64
    • kjcpp::ixmax
    • kjcpp::fx32
    • kjcpp::fx64
    • kjcpp::fx128
    • kjcpp::fxmax
    • kjcpp::class_type<char>
    • kjcpp::class_type<unsigned char>
    • kjcpp::class_type<char8_t>
    • kjcpp::class_type<char16_t>
    • kjcpp::class_type<char32_t>
    • kjcpp::class_type<wchar_t>

kjcpp::meric_static_cast can be used to convert:

  • real-kct to real-kct
  • real-number to real-kct
  • real-kct to real-number
  • real-number to real-number
