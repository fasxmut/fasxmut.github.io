kjcpp::fx128 - kjcpp::f128

kjcpp::fx128 is the 128 bit floating point class type. kjcpp::fx128 supports all basic floating point operators, and object-oriented class features. kjcpp::fx128 is longer than kjcpp::fx64, aka. double class type.

kjcpp::fx128 - kjcpp::f128, the class type of kjcpp::f128.

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::fx128 x = 1.225;
	kjcpp::print(x);
}
