kjcpp::ix8 - kjcpp::i8

kjcpp::ix8 is the 8 bit signed integral class type, it is "signed char" class type. All basic signed integral operators and bitwise operators are supported, and object-oriented class features are supported too.

kjcpp::ix8 - kjcpp::i8, the class type of kjcpp::i8.

  • .operator basic_type ()
  • .operator()()
  • .value()
  • .value_ref()

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::ix8 x = 25;
	kjcpp::print(kjcpp::ix16{x});
}
