kjcpp::sin

kjcpp::sin is a function object of class kjcpp::sin_class.

Calling Sig

Result type is always floating (auto-deduced), arg type is auto-deduced 

constexpr auto result = kjcpp::sin(value);

Result type is explicit set, arg type is auto-deduced 

constexpr auto result = kjcpp::sin.fn<result_type>(value);

Result type is explicit set, arg type is explicit set 

constexpr auto result = kjcpp::sin.fn<result_type, arg_type>(value);

Note

Note
  • The returned float type is one of kjcpp::f32, kjcpp::f64, kjcpp::f128, kjcpp::fmax, kjcpp::fx32, kjcpp::fx64, kjcpp::fx128, kjcpp::fxmax.
  • constexpr support.

Header 

#include <kjcpp/math.hpp>

Example 

#include <kjcpp/math.hpp>
#include <kjcpp/core.hpp>

int main()
{
	constexpr auto x = kjcpp::sin(std::numbers::pi/4);
	kjcpp::print(x);
}
