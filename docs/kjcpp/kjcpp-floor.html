kjcpp::floor

kjcpp::floor: get the floor value of kjcpp real meric value. The result type is the same as value type. 

template <kjcpp::real_meric real_type>
constexpr inline auto floor(const real_type & value);

c++ Example 

#include <kjcpp/math.hpp>

int main()
{
	kjcpp::fx64 a = 2.578;
	a = kjcpp::floor(a); // 2.0
}
