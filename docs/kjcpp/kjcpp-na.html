kjcpp::na

kjcpp::na is used to check if the comparison of two values have no result. Having result means the comparison result is one of <, <, >, >, ==; No result means not any one of them.

kjcpp::na returns true if no comparison result; returns false if having comparison result.

Their types must satisfy kjcpp::real_meric.

kjcpp::na is used to check if the comparison of two values have no result.

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::ux32 a = 7;
	kjcpp::fx32 b = 3.2;
	kjcpp::fx32 c = kjcpp::nan_v<kjcpp::fx32>;

	bool x = kjcpp::lt(a, b);
	bool y = kjcpp::gt(b, c);
	bool z = kjcpp::na(a, c);

	kjcpp::print(x, y, z); // false false true
}
