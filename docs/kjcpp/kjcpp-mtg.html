kjcpp::mtg

kjcpp::mtg, kjcpp::mtg64

kjcpp::mtg is the std::mersenne_twister_engine aka std::mt19937 object, a random number generator.

kjcpp::mtg64 is the 64 bit std::mersenne_twister_engine aka std::mt19937_64 object, a random number generator.

(std::mersenne_twister_engine) 

inline auto mtg = std::mt19937{std::random_device{}()};
inline auto mtg64 = std::mt19937_64{std::random_device{}()};

kjcpp::mtg and kjcpp::mtg64 are used as instant usage of random number generators.

c++ example 

#include <kjcpp/random.hpp>
#include <kjcpp/core.hpp>

int main()
{
	kjcpp::print(kjcpp::mtg(), kjcpp::mtg64());
}
