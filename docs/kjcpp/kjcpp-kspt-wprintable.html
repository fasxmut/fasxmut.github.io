kjcpp::kspt::wprintable

kjcpp::kspt::wprintable - Kjcpp Output Concepts

kjcpp::kspt::wprintable is a concept to constrain a wchar_t based printable types of std::wostream. The type must satisfy printable by std::wostream. Many basic types satisfy kjcpp::wprintable, such as int, float, bool. Specially, std::wstring satisfies kjcpp::kspt::wprintable, but std::string does not satisfy kjcpp::kspt::wprintable, it satisfies kjcpp::kspt::printable.

c++ example 

#include <kjcpp/core.hpp>

template <kjcpp::kspt::wprintable type1>
class fizz_buzz
{
public:
	fizz_buzz(const type1 & value1)
	{
		std::wcout << value1 << std::endl;
		kjcpp::wprint(value1);
	}
};

int main()
{
	fizz_buzz f1{123};
	fizz_buzz f2{L"Hello c++!"};
}
