.quo member function of kjcpp::class_type

kjcpp::class_type::quo

Method .quo of kjcpp::class_type is used to get the quotient of a value when calculating modulus value. It does not equal to division result.

Positive/Negative flag rules, a % b :

  • The flag of quotient is the same as a/b .
  • The flag of modulus is the same as a .

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::fx32 a = 33.3;
	kjcpp::print(a.quo(12);	// 3.0
}
