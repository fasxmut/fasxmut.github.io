kjcpp::kspt::printable - Kjcpp Output Concepts

kjcpp::kspt::printable concept is used to constrain a type is a std::ostream printable type. The type must satisfy prinable by std::ostream. Many basic types of the c++ language base satisfy kjcpp::printable, such as int, float, bool etc. the user-defined types can overload operator<< for std::ostream to satisfy kjcpp::kspt::printable. Specially, std::string satisfies kjcpp::kspt::printable; but std::wstring does not satisfy kjcpp::kspt::printable, it satisfies kjcpp::kspt::wprintable.

c++ exmaple 

#include <kjcpp/core.hpp>

template <kjcpp::kspt::printable type1>
class foo_bar
{
public:
	foo_bar(const type1 & value1)
	{
		std::cout << value1 << std::endl;
		kjcpp::print(value1);
	}
};

int main()
{
	foo_bar<int> f1{123};
	foo_bar<std::string> f2{"Hello, c++!"};
}
