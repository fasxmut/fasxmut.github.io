kjcpp::mean_t

kjcpp::mean_t - kjcpp::mean

kjcpp::mean_t is a type trait calculation to get the type from second template parameter or third template parameter depending on the condition is met or not.

kjcpp::mean_t is a compile-time conditional if-else template class for type identification.

kjcpp::mean_t has already been used in many cases of kjcpp library. It is proved very useful in practice. kjcpp::mean_t is a good type trait extracted from practice.

kjcpp::mean_t - kjcpp::mean 

template <bool condition, typename type1, typename type2>
class mean
{
public:
	using type = ???;
};

If condition is true, kjcpp::mean::type equals to type1; else kjcpp::mean::type equals to type2. 

template <bool condition, typename type1, typename type2>
using mean_t = kjcpp::mean<condition, type1, type2>::type;

Include 

#include <kjcpp/core.hpp>

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	using type1 = kjcpp::mean_t<true, kjcpp::i32, std::string>;
	static_assert(std::same_as<type1, kjcpp::i32>);

	using type2 = kjcpp::mean_t<false, kjcpp::i32, std::string>;
	static_assert(std::same_as<type2, std::string>);
}
