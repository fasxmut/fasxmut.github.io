kjcpp-mutex0~9

kjcpp::mutex0~9 are ten std::mutex objects, which are used as pre-created mutexes directly, then users do not have to declare mutexes any more. These kjcpp::mutexes save us lots of jobs.

To guard same resources against different threads, the same mutex should be used.

c++ example 

#include <kjcpp/thread.hpp>
#include <kjcpp/core.hpp>

int main()
{
	auto f1 = kjcpp::async(
		[]
		{
			for (kjcpp::ix32 i=0; i<10; ++i)
			{
				std::unique_lock lock{kjcpp::mutex0};
				kjcpp::print("i is ", i);
			}
		}
	);
	auto f2 = kjcpp::async(
		[]
		{
			for (char i='A'; i<='G'; ++i)
			{
				std::unique_lock lock{kjcpp::mutex0};
				kjcpp::print("char i is ", i);
			}
		}
	);
}
