kjcpp::u8str_size

kjcpp::u8str_size - Kjcpp

kjcpp::u8str_size - A function object to get the length of a u8 string: get the character counter.

kjcpp::str_size_class

The function object is the object of instantiated template class kjcpp::str_size_class .

Get String Size - How

Get Array String Size

It gets string size by template type traits.

Get const type_xt * or type_xt * String Size

It gets string size by searching until '\0' is found.

It works on both string literal or string made by new.

For kjcpp::str_size, type_xt is char; for kjcpp::wstr_size, type_xt is wchar_t, ..., etc.
