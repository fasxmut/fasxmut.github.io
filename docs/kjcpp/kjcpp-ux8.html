kjcpp::ux8 - kjcpp::u8

kjcpp::ux8 is the 8 bit unsigned integral class type, it is "unsigned char" class type. kjcpp::ux8 supports all basic unsigned integral operators and bitwise operators, and it supports object-oriented class features.
