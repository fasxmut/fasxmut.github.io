kjcpp::round

kjcpp::round: get the round value of kjcpp real meric value. The result type is the same as the value type. 

constexpr inline auto round(const type1 & value);

c++ Example 

#include <kjcpp/math.hpp>

int main()
{
	kjcpp::fx32 d = 32.23f;
	d = kjcpp::round(d); // 32.0f
	kjcpp::fx32 e = 32.6f;
	e = kjcpp::round(e); // 33.0f
}
