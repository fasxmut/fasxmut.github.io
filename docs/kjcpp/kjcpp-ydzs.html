kjcpp::ydzs

kjcpp::ydzs, aka yield-this-thread, is used to yield this thread, which will call std::this_thread::yield. kjcpp::ydzs() is a shorthand of std::this_thread::yield().

c++ example 

#include <kjcpp/thread.hpp>
#include <kjcpp/core.hpp>

int main()
{
	auto f1 = kjcpp::async(
		[]
		{
			for (kjcpp::ix32 i=0; i<10; ++i)
			{
				kjcpp::print(i);
				kjcpp::ydzs();
			}
		}
	);
}
