kjcpp::fact

kjcpp::fact calculates the factorial result of value.

Calling Sig: 

constexpr auto result = kjcpp::fact(unsigned_value);

constexpr auto result = kjcpp::fact.fn<return_type>(unsigned_value);

constexpr auto result = kjcpp::fact.fn<return_type, unsignned_arg_type>(value);

kjcpp::fact(???) :

  • If result is less than value, it indicates the result is overflow. When that happens, the result is 3 (There is no any unsigned integral factorial result is 3, so kjcpp::fact uses 3 to indicate overflow).
  • The returned type is same as value type, but the value type must satisfy unsigned. That's one of kjcpp::u8, kjcpp::u16, kjcpp::u32, kjcpp::u64, kjcpp::umax, kjcpp::ux8, kjcpp::ux16, kjcpp::ux32, kjcpp::ux64, kjcpp::uxmax.
  • constexpr support.

kjcpp::fact.fn<return_type>(???) :

kjcpp::fact.fn<return_type, unsigned_arg_type>(???) :

  • The argument type must be still unsigned integral type, the result is converted to return_type.

Header 

#include <kjcpp/math.hpp>

Example 

#include <kjcpp/core.hpp>
#include <kjcpp/math.hpp>

int main()
{
	// arg_type is kjcpp::u32
	constexpr auto result1 = kjcpp::fact(8u);

	// arg_type is kjcpp::u32, result is converted to kjcpp::f32
	constexpr auto result1 = kjcpp::fact<kjcpp::f32>(8u);

	// arg_type is kjcpp::ux32, result is converted to kjcpp::ix32
	constexpr auto result2 = kjcpp::fact<kjcpp::ix32, kjcpp::ux32>(8);

	kjcpp::print(result1, result2, result3);
}
