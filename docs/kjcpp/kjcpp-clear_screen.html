kjcpp::clear_screen

kjcpp::clear_screen is an object to clear the screen characters. It does not work on all environments. It works on linux at least.

Calling Sig: 

kjcpp::clear_screen();

c++ Example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::clear_screen();
}
