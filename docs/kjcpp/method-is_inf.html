.is_inf method

.is_inf member function of kjcpp::class_type

kjcpp::class_type::is_inf

Method .is_inf of kjcpp::class_type is used to check if the object value is infinity.

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::fx64 a = kjcpp::fx64_max;
	const kjcpp::fx64 b = a/0.123;
	const kjcpp::fx64 c = -b;
	kjcpp::print(a.is_inf(), b.is_inf(), c.is_inf());	// false true true
}
