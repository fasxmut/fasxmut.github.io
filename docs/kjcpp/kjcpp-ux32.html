kjcpp::ux32

kjcpp::ux32 - kjcpp::u32

kjcpp::ux32 is the unsigned 32-bit integral type that can be used as basic unsigned int; it is class type: kjcpp::class_type, which has overloaded almost every operations of the basic integral types; if not, it will be added in the future.

kjcpp::ux32 - kjcpp::u32, the class type of kjcpp::u32.

Except the basic integral operations, kjcpp::ux32 has some advanced class features, such as some member functions, and some features that are not possible for basic integral types.

  • operator basic_type ()
  • operator()()
  • .value()
  • .value_ref()

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::ux32 rgba  = 0x123456ff;
	constexpr kjcpp::ux32 rgb = (rgba >> 8);
	static_assert(rgb = 0x00123456);
}
