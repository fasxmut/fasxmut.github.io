kjcpp::fx32 - kjcpp::f32

kjcpp::fx32 is 32-bit floating point class type. It is c++ class, so kjcpp::fx32 is object-oriented basic floating type.

kjcpp::fx32 - kjcpp::f32, the class type of kjcpp::f32.

kjcpp::fx32 can be initialized by a floating number, as it has converting constructor to initialize the object: 

kjcpp::fx32 x = 3.14314f;
kjcpp::fx32 y{4.13f};

kjcpp::fx32 fulfills all basic floating number operarors, and it also has operator<< to do printing: 

kjcpp::fx32 x = 1.234f;
std::cout << x << std::endl;

kjcpp::fx32 has three member functions which make kjcpp::fx32 to be converted to basic floating number:

  • .operator basic_type ()
  • .operator()()
  • .value() 
kjcpp::fx32 x = 8.9f;
kjcpp::f32 a = x;
kjcpp::f32 b = x();
kjcpp::f32 c = x.value();

kjcpp::fx32 has a special member function to get the underlying value ref:

  • .value_ref()

kjcpp::fx32 c++ example 

#include <kjcpp/core.hpp>

int main()
{
	kjcpp::fx32 x = 1.1f;
	kjcpp::fx32 y = x*(x+1);
	kjcpp::print(x, y); // 1.1 2.31
}
