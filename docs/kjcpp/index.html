Kjcpp Library

Kjcpp

Kjcpp is an Evolving Sekos Versatile c++ library that makes things from unique view, it creates some general helpful APIs when evolving. Kjcpp library is object-oriented, class-oriented, template-oriented, concept-oriented and general-oriented.

Kjcpp is cross-platform, working on all modern c++ compliant (at least c++23) operating systems.

Evolving Sekos Versatile

Kjcpp is evolving. Once new idea is accepted, it will be added to the library quickly. Kjcpp supports many different functionals.

object-oriented, class-oriented

The object-oriented programming and classes are the first citizen of kjcpp. Each API is created from the beginning of creating a class.

template-oriented

Kjcpp uses classes of templates, meta-templates, type-traits. The templates that are used by kjcpp types are the type system of kjcpp.

concept-oriented

Kjcpp uses c++20 concepts to constrain its templates, for writing correct template parameters, and reducing compiling messages.

general-oriented

Kjcpp is general-oriented, provding the general APIs, it does not provide special doamain APIs. Kjcpp is not for special domains, it is generic for all domains. Special domains still need special c++ libraries.
