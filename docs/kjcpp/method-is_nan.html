.is_nan member function of kjcpp::class_type

kjcpp::class_type::is_nan

Method .is_nan of kjcpp::class_type is used to check if the object value is NaN value. NaN value means the value can not compare by using any one of <, >, ==, !==, <, > .

c++ example 

#include <kjcpp/core.hpp>
#include <limits>

int main()
{
	constexpr kjcpp::fx32 a = std::numeric_limits<kjcpp::f32>::quiet_NaN();
	static_assert(a.is_nan());
	constexpr kjcpp::fx32 b = kjcpp::fx32_max;
	static_assert(! b.is_nan());
}
