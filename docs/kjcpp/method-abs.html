.abs method

.abs member function of kjcpp::class_type

kjcpp::class_type::abs

Method .abs of kjcpp::class_type can be used to get the absolute value of integral or floating value of kjcpp::class_type.

c++ example 

#include <kjcpp/core.hpp>

int main()
{
	constexpr kjcpp::ix32 a = 254;
	constexpr kjcpp::fx64 b = 542.445;
	constexpr kjcpp::ix32 c = -254;
	constexpr kjcpp::fx64 d = -542.445;
	kjcpp::print(a.abs(), b.abs(), c.abs(), d.abs());	// 254 542.445 254 542.445
}
