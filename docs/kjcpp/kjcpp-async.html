kjcpp::async

kjcpp::async is used to create asynchronous std::future thread. Unlike std::async, kjcpp::async is always in async mode: std::launch::async. If the operating system or runtime environment does not support asynchronous thread model, kjcpp::async prefers crash, not continue creating a non-asynchronous thread.

kjcpp::async is based on std::async, that kjcpp::async will call std::async.

c++ example 

#include <kjcpp/thread.hpp>
#include <kjcpp/core.hpp>

int main()
{
	std::future<void()> f1 = kjcpp::async([] {kjcpp::print("Hello c++!");});
	auto f2 = kjcpp::async([] {kjcpp::print("c++ Hello!");});
}
