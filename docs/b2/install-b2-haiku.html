B2 Build Haiku, Build and Install

B2 Build Home:

https://www.bfgroup.xyz/b2

On Haiku OS:

  • Requires gnu c++ compiler: gcc 
> g++ --version
  • Get b2 source: 
> git clone --depth=1 https://github.com/bfgroup/b2
  • Build B2 
> cd b2
> ./bootstrap.sh
  • Install B2 
> ./b2 install --prefix=/boot/home/sand
  • Add /boot/home/sand/bin to your system bin path.
  • Create and Build c++ project

Init project: 

> mkdir my-project
> cd my-project
> kak hello-world.cpp
> kak jamroot

hello-world.cpp : 

#include <iostream>

int main()
{
	std::cout << "Hello, c++!" << std::endl;
}

jamroot : 

exe hello-world : hello-world.cpp : : <cxxstd>23 ;

Build project: 

> b2

...updated 6 targets...

Run program: 

> ./bin/gcc-13/debug/cxxstd-23-iso/hell-world
Hello, c++!
