decoder.result_size() is method of the class azvzl::az_decoder object. It returns the decoded data size of previously calling decoder.decode(...) .

Calling Sig: 

auto size = decoder.result_size();

If the decoding of calling decoder.decode(...) has error, it returns zero.

It is used to get the decoded data size of decoder.decode(...) (overload-1), for (overload-2), it returns zero. (Read decoder.decode(...))
