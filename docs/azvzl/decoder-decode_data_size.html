decoder.decode_data_size is a method of class azvzl::az_decoder object. It is used to pre-decode data output data section size of the to be decoded data.

The method is const, which will affect any state.

This method is used by interally, and can be used by caller for pre-fetch the output data section size.

Calling Sig: 

auto size = decoder.decode_data_size(
	input__,		// const pointer, the input data
	input_size__,		// const, the input data size
	prop_size__		// const, the input props size
);
