decoder.result() is the method of the azvzl::az_decoder object. It returns preivously decoded result of calling decoder.decode(...). 

auto result = decoder.result();

If the previous decoded result is invalid, it returns nullptr.

It returns the decoded result of decoder.decode(...) (overload-1). For decoder.decode(...) (overload-2), it returns nullptr. (Read decoder.decode(...))
