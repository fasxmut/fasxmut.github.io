azvzl::az_decoder is a class of the azvzl c++ library. It is used to create decompression object. It is better to create one object of it for one program and call methods many times.

Calling Sig: 

azvzl::az_decoder decoer{};

This default constructor initializes the class member variables with best values.

include:

azvzl/az_decode.hpp 

#include <azvzl/az_decode.hpp>
