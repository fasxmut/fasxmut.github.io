How to access cppfx.i2p

From today (12/05/2025), cppfx.xyz has been migrated to cppfx.i2p,
and set cppfx.i2p as master site.
The updates will be applied to cppfx.i2p first, then cppfx.xyz.

How to access cppfx.i2p

step-1: Install i2p client

Install i2p client, such as i2pd,

===>

Run i2pd,

Run in foreground: 

i2pd

Run in background: 

i2pd --daemon

===>

Wait i2pd peer data and addressbook synchronization,
it might take a quick short time or a long time, depending on your network environment.

===>

Watch synchronization status

For i2pd,
Open http://127.0.0.1:7070
Click I2P Tunnels,

Under Client Tunnels,
you will see HTTP Proxy and SOCKS Proxy,
just click them and watch Inbound tunnels and Outbound tunnels .

step-2: Configure web page browser

Take falkon for example,

Open falkon main menu

=>

click "Preferences"

=>

click "Browsing"

=>

click "Proxy Configuraton"

=>

select "Manual configuration"

=>

(You can choose http proxy or socks proxy now)
(If you are using i2pd:)

HTTP		127.0.0.1		4444

SOCKS5		127.0.0.1		4447

Then restart falkon.

step-3: Access cppfx.i2p

After restarting falkon, just open any one of the domains:

cppfx.i2p
