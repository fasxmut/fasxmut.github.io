vp::instance

vp::instance is a c++ class that is derived from vk::Instance, all vk::Instance methods and features are available in vp::instance, and vp::instance can create platform windows, macos, linux x11, xcb, wayland related instance directly. vp::instance's object can be referenced by native type vk::Instance. virtual is supported by vp::instance.

features

  • platform related instance

vp::instance can create platform related instance vk::Instance directly: windows, macosx, x11, xcb, wayland

  • native referenced

vp::instance's object can be referenced by native type: vk::Instance

  • derived from vk::Instance

vp::instance is derived from vk::Instance, all methods and features in vk::Instance are available in vp::instance .

  • virtual

virtual destructor, virtual inheritance

  • using base_type = ... ;

class vp::instance 

class instance: public vk::Instance;

instance.construct_win32();

vp::instance object can create windows vk::Instance directly.

instance.construct_macos();

vp::instance object can create mac os x vk::Instance directly.

instance.construct_x11();

vp::instance object can create linux x11 vk::Instance directly.

instance.construct_xcb();

vp::instance object can create linux xcb vk::Instance directly.

instance.construct_wayland();

vp::instance object can create linux wayland vk::Instance directly.

Native Referenced

vp::instance is derived from vk::Instance, vp::instance object can be referenced by native type vk::Instance with polymorphism. 

vp::instance instance;
vk::Instance & native_ref = instance;

virtual

destructor is virtual, polymorphism is supported.

example

c++ short example 

#include <vpcpp/vpcpp.hpp>

int main()
{
	vp::instance instance;
	instance.construct_win32();
	vk::Instance & native_ref = instance;
}
