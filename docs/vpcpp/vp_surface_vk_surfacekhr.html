vp::surface

vp::surface is a c++ class that is derived from vk::SurfaceKHR in c++ vulkan header vulkan.hpp , all vk::SurfaceKHR features are available in vp::surface. vp::surface provides both native and ex_native references.

features

  • native and ex_native references

vp::surface's object can be referenced by native ( vk::SurfaceKHR ) and ex_native ( VkSurfaceKHR ).

  • vp::surface is derived from vk::SurfaceKHR

all methods in vk::SurfaceKHR are available in vp::surface .

  • virtual

vp::surface class supports virtual destructor and virtual inheritance.

class vp::surface 

class surface: public vk::SurfaceKHR;

type traits

  • using base_type = ... ;
  • using self_type = ... ;
  • using native_type = ... ;
  • using ex_native_type = ... ;

operator ref operator copy

vp::surface's obejct can be referenced by native type: vk::SurfaceKHR and ex_native type: VkSurfaceKHR . This makes vk::SurfaceKHR to be helpful in many cases.

derived from vk::SurfaceKHR

Because vp::surface is derived from vk::SurfaceKHR, all methods and features of vk::SurfaceKHR are available in vp::surface .

virtual

Destructor is virtual, supports c++ polymorphism.

example

c++ short example 

#include <vpcpp/vpcpp.hpp>

int main()
{
	vp::surface surface;
	vk::SurfaceKHR & native_ref = surface;	// polymorphism
	VkSurfaceKHR & ex_native_ref = surface;
}
