Nirtcpp Backends

Nirtcpp bakends can be divided into two groups: driver backends and device backends, just like irrlicht.

  • Driver Backends
  • Device Backends

Driver backends include the following backends:

  • nirt::video::EDT_NULL
  • nirt::video::EDT_SOFTWARE
  • nirt::video::EDT_BURNINGSVIDEO
  • nirt::video::EDT_DIRECT3D9
  • nirt::video::EDT_OPENGL

Device backends include the following backends:

  • nirt::EIDT_WIN32
  • nirt::EIDT_X11
  • nirt::EIDT_OSX
  • nirt::EIDT_SFML
  • nirt::EIDT_FRAMEBUFFER
  • nirt::EIDT_CONSOLE
  • nirt::EIDT_BEST

There are some more backends that are not listed here, they are not guaranteed working and are not tested, have less usage cases, and might be removed in the future.
