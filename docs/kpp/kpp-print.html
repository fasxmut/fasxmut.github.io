kpp::print

---- kpp::print << Output << Kpp c++ ----

kpp::print is a printer object of kpp c++ library. kpp::print is re-designed from scratch.

kpp::print prints many different values by one object.

Features:

  • Multiple value printer
  • No formaters
  • thread-safe

kpp::print usage Sig: 

kpp::print(value1, value2, ...);

include 

#include <kpp/print.hpp>

c++ example 

#include <kpp/print.hpp>

int main()
{
	kpp::print("Hello World, c++!", true);
}
