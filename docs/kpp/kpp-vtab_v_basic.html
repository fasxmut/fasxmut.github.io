kpp::vtab_v_basic

---- kpp::vtab_v_basic << Empty String Constants << Basic String Utils << Type System << Kpp c++ ----

kpp::vtab_v_basic is the c++ vtab character of kpp c++ library. It is a class template alias, the parameter of which is one of character types:

char, wchar_t, char8_t, char16_t, char32_t,

c++ example 

constexpr inline auto vtab__ = kpp::vtab_v_basic<wchar_t>;
kpp::same_v_assert<vtab__, L'\v'>{};
