kcpp::dice

kcpp::dice the c++ dicing output stream object of kcpp output stream class.

kcpp::dice prints many values once, it is a multiple value printer.

kcpp::dice works fro low level to high level.

include 

#include <kcpp/io.hpp>

c++ example 

#include <kcpp/io.hpp>

int main()
{
	kcpp::dice('a', "cat", 23);
/*
a cat 23
*/
}
